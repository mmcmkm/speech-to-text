# 音声文字起こしツール

Gemini APIを使用して音声入力から文字起こしを行い、「えーと」「あー」などの躊躇いや曖昧な表現を自動的に除去するツールです。

## 機能
- 音声録音機能
- Gemini APIを使用した音声文字起こし
- 口癖や躊躇表現の自動除去
- クリップボードへの自動コピー

## 必要条件
- Python 3.8以上
- Gemini API キー（環境変数 `GEMINI_API_KEY` に設定）

## インストール方法

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py
```

1. アプリケーションを起動
2. 録音ボタンをクリックして録音開始
3. 停止ボタンをクリックして録音終了
4. 文字起こし結果が表示され、自動的にクリップボードにコピーされます
